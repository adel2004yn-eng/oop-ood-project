# oop-ood-project

E-Shop Console Application (Java)
A modular console-based e-commerce system implemented in Java, demonstrating strong object-oriented design, role-based architecture, and design patterns.
The system is built around a User abstraction, with specialized user roles such as Buyer, Seller, and Manager, each implementing role-specific behavior while sharing common user functionality.

Key Features:

Abstract User base class with role-based inheritance
Buyer, Seller, and Manager system roles
Product catalog organized by categories
Shopping cart functionality
Product creation via Factory Method pattern
Console-based UI separated from business logic
Clean, extensible architecture
Technologies & Concepts
Java
Object-Oriented Programming (Inheritance, Polymorphism)
Design Patterns (Factory Method)
Collections & Data Structures
Separation of Concerns
